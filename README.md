# Intro-to-ML_WIDS

This Winter in Data Science Project under the Analytics Club was mentored by [Kushagra](https://github.com/Kushagra-o7) and [Saksham](https://github.com/PsychoX21)

## Contents

-   Week-1 : Learning basics of Python and required libraries such as Numpy, Matplotlib, Pandas and then assignments based on those.
-   Week-2 : Learning to implementing Gradient Descent, Linear Regression, Logistic Regression, and other related concepts followed by assignments on both linear and logistic regression.
-   Week-3 : Learning the basics of Artificial and Convolutional Neural Networks and implementing them in assignments.
