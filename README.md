# Intro-to-ML_WIDS

This Winter in Data Science Project under the Analytics Club was mentored by [Kushagra](https://github.com/Kushagra-o7) and [Saksham](https://github.com/PsychoX21)

## Contents

-   Week-1 : Learning basics of Python and required libraries such as Numpy, Matplotlib, Pandas and then assignments based on file reading, implementing algorithms, implementing overloads, and processing and representing data with pandas, numpy and matplotlib.
-   Week-2 : Learning to implementing Gradient Descent, Linear Regression, Logistic Regression, and other related concepts followed by assignments where i implemented linear regression on the given student performance data and logistic regression for classfying the given data.
-   Week-3 : Learning the basics of Artificial and Convolutional Neural Networks and implementing a simple neural net based on given data in ANN part and hardwritten digit detection in the CNN part.
